You can compile plugins within ImageJ using either the
Plugins>Compile and Run command or the File->Compile and Run
command in ImageJ's built in editor (Plugins>Edit).

The plugins contained in Input-Output>QuickTime_Plugins.jar
are preinstalled as the File>Import>Using QuickTime...,
File>Import>Video, File>Import>QuickTime Player.. and
File>Save As>QuickTime Movie... commands.

The following third party plugins are included:

Analyze Reader/Writer
   File>Import>Analyze..., File>Save As>Analyze...
   http://rsbweb.nih.gov/ij/plugins/analyze.html
   
Biorad Reader
   Plugins>Input-Output>Biorad Reader
   http://rsbweb.nih.gov/ij/plugins/biorad.html

Color Histogram
   Analyze>Tools>Color Histogram    
   http://rsbweb.nih.gov/ij/plugins/color-histogram.html

LSM Reader
   Plugins>Input-Output>LSM Reader
   http://rsbweb.nih.gov/ij/plugins/lsm-reader.html
